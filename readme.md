# Trip Management App

Record trips taken and view them on map.\
A debug apk is present in [apk folder](./apk)

## Configs and Libraries used

-Room\
-MVVM\
-Data Binding\
-Coroutines\
-Fused Location Provider\
-LiveData\
-Google Play Services\
-Google Maps
